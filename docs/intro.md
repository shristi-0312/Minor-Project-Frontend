# Documentation

Let's discover **our project and it's features**.

## Getting Started

Get started by **Checking out the Dashboard Feature**.


### What you'll need

- [Node.js](https://nodejs.org/en/download/) version 16.14 or above:
  - When installing Node.js, you are recommended to check all checkboxes related to dependencies.

## Doing a Quick Check

Runs the required scans using out toolbar in  **Dashboard**.

The classic template will automatically be added to your product after you run the command:

```bash
npm init docusaurus@latest my-website classic 
```

You can type this command into Command Prompt, Powershell, Terminal, or any other integrated terminal of your code editor.

The command also installs all necessary dependencies you need to run our Minor Project.
